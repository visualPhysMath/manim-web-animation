# Manim Web Animation

This is a minimal setup that renders a Manim animation with GitHub Actions and publishes it with GitHub Pages.

## Usage

1. Put these files in a GitHub repository.
2. Open `Settings` -> `Pages` -> `Build and deployment`.
3. Set `Source` to `GitHub Actions`.
4. Push to the `main` branch.
5. When the `Render Manim and Deploy Pages` workflow succeeds, the animation is available on the Pages URL.

## Editing the animation

Edit the `WebAnimation` class in `main.py`.
